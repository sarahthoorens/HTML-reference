HTML Reference Sheet

Common HTML Elements
Tag Name Description
<head> Head Contains the document's metadata, not meant for human-visible information. Stylesheets and other resources may be linked here.
<title> Page Title Nests within the <head> element; the title will be displayed in a browser's tab.
<body> Body Contains all visible content of the webpage.
<h1>, <h2>...<h6> Headings Heading styles that display the most emphatic style (<h1>) and end with the least emphatic (<h6>).
<p> Paragraph A paragraph or block of text used to structurally group related content of a document.
<div> Content Division A container for content best used for block separation.
<span> Content Span A container for content best used for inline text.
<a> Anchor Used along with targeted href attribute, it wraps content and hyperlinks to href destination.
<ul> Unordered List Creates an unordered, bulleted list in a block. Each subsequent listed item must be wrapped with <li> tags.
<ol> Ordered List Creates an ordered, numbered list in a block. Each subsequent listed item must be wrapped with <li> tags.
<section> Section Defines chapters, headings or grouped themes of a page.
<article> Article Holds content that makes sense on its own, lie articles, blogs, comments, magazines; could be a combination of text, images, audio, etc. Articles are typically nested within a Section, but the inverse could happen, too.
<aside> Aside Holds related content, but that which is not required to understand the main content, such as bibliographies, comments, end notes or editorial sidebars.
